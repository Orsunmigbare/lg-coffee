# lg-calc
<img alt="tested with mocha and chai" src="https://img.shields.io/badge/tested-with_mocha_and_chai-ff69b4.svg" />

A utility that accepts an object and returns an integer. 

## Setup
```bash
$ npm i lg-calc
```

### Use

```js
import calc from 'lg-calc';

calc({
  one: {
    price: 287,
    qty: 5,
  },
  two: {
    price: 233,
    qty: 3,
  },
}); // returns 2134
```
